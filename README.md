## Lesson Objectives

Learners will understand:

- Various data encoding formats

  - XML
  - JSON
  - Thrift
  - Protocol Buffers
  - Avro
  - Parquet
  - ORC
  - Arrow

- Data flow modes
  - Databases
  - Service calls
    - REST API
    - GraphQL
    - RPC
  - Asynchronous message passing
    - Message queue
    - Pub-sub

Learners will be able to:

- Encode and decode data in various formats
- Exchange data via RPC
